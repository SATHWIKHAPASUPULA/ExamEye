# ExamEye
ExamEye is an intelligent real-time cheating detection system designed for online examination environments. It combines advanced computer vision techniques, deep learning models, and knowledge graph-based reasoning to monitor student behavior and detect suspicious activities automatically.
The system enhances academic integrity by identifying actions such as mobile phone usage, looking away, presence of multiple persons, and abnormal behavior during exams.
Key Features
Real-Time Monitoring using webcam/video input
Object Detection (YOLOv8) – detects mobile phones, persons, etc.
Gaze Tracking & Head Pose Estimation
Knowledge Graph (Neo4j) for behavior modeling
Rule-Based Expert System for decision making
Session Summary & Alerts
Flask Web Interface for interaction
